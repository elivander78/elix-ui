## [0.2.1] - 2024-12-19

### Changed
- Bump version to 0.2.1

## [0.1.0-beta] - 2024-12-19

### Added
- Initial release of @elivander/elix-ui
- Vue 3 UI component library with theming and responsive design
- Comprehensive component library including:
  - Button, ButtonGroup
  - Input, Textarea
  - Card (with CardHeader, CardBody, CardFooter)
  - Select
  - Modal, Drawer
  - Switch, Checkbox, Radio
  - Badge, Tag, Avatar
  - Divider
  - Tabs, TabPane
  - Form, FormItem
  - List, ListItem
  - Steps, Step
  - Timeline, TimelineItem
  - Container, Row, Col
  - Stack
  - Header, Sidebar
  - Spinner
  - Skeleton
  - Progress
  - Pagination
  - Table
  - Alert
  - Empty
  - Result
  - Statistic
  - Tooltip, Popover
  - Dropdown
  - Collapse, CollapseItem
  - DatePicker, BaseDatePicker
  - Upload
  - Notification
- Theming system with light and dark themes
- CSS variables for customization
- Responsive design utilities
- TypeScript support
- Tree-shakeable imports
- Documentation site with examples
- GitHub Pages deployment setup
- Theme persistence

### Fixed
- GitHub icon styling to use stroke instead of fill
- GitHub button styling (round shape)
- GitHub Pages SPA routing (404.html)
- Mixed export warnings in rollup config
- Component styles and accessibility improvements

### Changed
- Updated component styles for better consistency
- Enhanced README with deployment instructions
- Improved dependency management
